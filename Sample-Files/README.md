# Input Files

Welcome to the **Input Files** section.

This folder contains sample input files used for learning, practicing, testing, and building mini projects.

These files are reused across multiple Python concepts such as File Handling, CSV, JSON, XML, Excel, and Data Engineering exercises.

---

## 📂 Folder Structure

```text
Input-Files/
│
├── txt/
├── csv/
├── json/
├── xml/
├── parquet/
├── excel/
├── images/
└── others/
```

---

## 📁 Folder Description

### 📄 txt
Sample text files used for practicing file handling operations.

### 📊 csv
CSV datasets used for data processing, analysis, and ETL exercises.

### 🌐 json
JSON files used for parsing, serialization, APIs, and configuration.

### 🏷️ xml
XML files for learning XML parsing and data transformation.

### 🗄️ parquet
Parquet files used in Data Engineering and PySpark examples.

### 📈 excel
Excel workbooks used for reading and writing spreadsheet data.

### 🖼️ images
Images used for image processing or file handling examples.

### 📦 others
Any additional input files that don't belong to the above categories.

---

## 🎯 Goal

- Store reusable input files for hands-on practice.
- Keep code and input data organized.
- Reuse the same datasets across examples, exercises, and mini projects.
- Simulate real-world data used in Data Engineering workflows.
